# Beverly Institute of Bitter Refreshment

An unofficial, satirical fan site for Beverly, the bitter Italian aperitif soda
that Americans mostly encounter in tiny tasting cups and unforgettable facial
expressions.

## What is here?

- A static multi-page HTML/CSS/JS site
- Original generated artwork saved in `assets/images`
- Real source links for Beverly tasting locations
- Completely made-up Beverly mythology, recipes, travel protocol, and hymn

This is not affiliated with The Coca-Cola Company, Disney, TSA, Italy, grapefruit,
or any official soda authority.

## Local preview

Open `index.html` directly in a browser, or run a tiny local server from this
folder:

```sh
python3 -m http.server 8765
```

Then visit `http://127.0.0.1:8765/`.
